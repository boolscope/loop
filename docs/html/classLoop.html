◆ isReady()

bool Loop::isReady ( bool  reset = false)

Readiness check.

Parameters
[in]resetif true, the timestamp indicator will be reset automatically when the object reaches the ready state.
Return values
trueif the set interval is exhausted and the object is ready;
falseotherwise.

It is convenient for quick operations:

if (loop.isReady(true)) {
// a fast code here ...
}

But for complex code that requires a long time for its execution, better in manual mode reset of the timestamp:

if (loop.isReady()) {
// a slow code here ...
loop.resetTime();
}
